# The Gini–Bayes Connection

**The CAP Slope as Bayes' Theorem, with Applications to Weight of Evidence, Somers' *D*, and Calibration.**

A working paper by Denis Burakov.

The slope of the cumulative accuracy profile (CAP) is Bayes' theorem in cumulative
coordinates: the standardized PD `p(D|x)/p_D` is exactly the slope `dF(x|D)/dF(x)`. From that
one identity the paper shows that

- the **Accuracy Ratio, Somers' *D*, and `2·AUC−1`** are one number computed three ways
  (with the exact `AR = (2A−1)/(1−p_D)`, not the `2A−1` shortcut);
- the **weight of evidence** is the log-ratio of the bad and good CAP slopes, and the
  **information value** is the Jeffreys divergence of the two slope profiles;
- run in *comparison mode* (empirical vs. model-implied CAP), the identity is the
  **reliability diagram in cumulative coordinates**, with the gap between the two Ginis as a
  one-number calibration summary;
- a worked **kernel-density example** recovers the weight-of-evidence and calibration curves
  from the class-conditional densities alone.

The slope identity traces to Falkenstein, Boral and Carty (2000, Moody's RiskCalc) and was
formalized by van der Burgt (2008, 2019); this paper derives it straight from Bayes and uses it
as a calibration diagnostic.

## Contents

| File | What it is |
|------|------------|
| `gini-bayes.pdf` | The paper. |
| `gini-bayes.tex` | LaTeX source (compile with XeLaTeX; figures in pgfplots/TikZ). |
| `scripts/five_band_example.py` | Discrete five-band example: Bayes reweighting, AR three ways, WOE/IV, calibration gap. |
| `scripts/continuous_kde.py` | Continuous KDE example: recovers WOE and the calibration curve from class-conditional densities; writes the figure data. |
| `data/*.dat` | pgfplots data for the continuous-example figures (regenerated by `continuous_kde.py`). |

## Reproduce

Dependencies are managed with [uv](https://docs.astral.sh/uv/):

```bash
uv run python scripts/five_band_example.py     # discrete example
uv run python scripts/continuous_kde.py        # continuous KDE example + figure data
```

Build the paper (needs XeLaTeX and the JetBrains Mono / Latin Modern Math fonts):

```bash
xelatex gini-bayes.tex && xelatex gini-bayes.tex
```
